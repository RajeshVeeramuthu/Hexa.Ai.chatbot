# Hexa.Ai.chatbot.
Chatbot.
This is a chatbot project done by RajeshVeeramuthu. 
It uses HTML 5, CSS 3, Bootstrap 5, and JavaScript.
This project is in the initial stage and it is undergoing maintenance, but it can handle simple queries.
For Example:
              1.Hello.
              2.open youtube.
              3.open google.
              4.time 
              5.exit
